MODEL TO SPRITE SHEET CONVERTER PRO

Model To Sprite Sheet Converter Pro is a professional Unity Editor tool that converts 3D models into high-quality 2D sprite sheets, built to dramatically speed up your 2D asset pipeline. Requires Unity 2022.3 LTS or higher.

Enjoy a modern, unified interface with three powerful modes — Single Model, Animation Preview, and Batch Processing — all in one clean window for a fast, workflow-based experience.

Supercharge animations: render any clip to a sprite sheet, live-preview in a dedicated tab, and even assign AnimationClips from outside the model. Export the currently previewed animation directly with one click.

Dial in the perfect shot with 6 camera presets (Isometric, Top-Down, etc.) plus precise Zoom/Position/Rotation controls. Generate from 1 to 32 directions for true 8-way movement. Style your output with 20+ shaders & post-effects (Cartoon, Pixel, Neon, Glitch, CRT) with anti-aliased, pro-lit results.

Ship-ready exports: PNG/TGA with transparency, comprehensive JSON metadata, auto-sliced sprites and even Animator Controller generation. Scale up with folder-level batch processing and a clean C# API for automation.
